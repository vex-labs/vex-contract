This file contains a list of example commands to be run in near-cli-rs
Remember to get cusd tokens on your own account

Delete account 

Create account 
cargo near create-dev-account use-specific-account-id meowwwww.testnet autogenerate-new-keypair save-to-legacy-keychain network-config testnet create

send ft to contract account to register
near tokens pivortex.testnet send-ft cusd.fakes.testnet meowwwww.testnet amount-ft '1 cUSD' prepaid-gas '100.0 Tgas' attached-deposit '1 yoctoNEAR' network-config testnet sign-with-legacy-keychain send

deploy and init
cargo near deploy meowwwww.testnet with-init-call init json-args '{"admin": "pivortex.testnet"}' prepaid-gas '100.0 Tgas' attached-deposit '0 NEAR' network-config testnet sign-with-legacy-keychain send

change_admin
near contract call-function as-transaction meowwwww.testnet change_admin json-args '{"new_admin": "icespice.testnet"}' prepaid-gas '100.0 Tgas' attached-deposit '0 NEAR' sign-as pivortex.testnet network-config testnet sign-with-legacy-keychain send

get_admin
near contract call-function as-read-only meowwwww.testnet get_admin json-args {} network-config testnet now

create_match
near contract call-function as-transaction meowwwww.testnet create_match json-args '{"game": "Overwatch", "team_1": "a team", "team_2": "b team", "in_odds_1": 1.4, "in_odds_2": 2.9, "date": "28/08/2024"}' prepaid-gas '100.0 Tgas' attached-deposit '0 NEAR' sign-as pivortex.testnet network-config testnet sign-with-legacy-keychain send

bet
near contract call-function as-transaction cusd.fakes.testnet ft_transfer_call json-args '{"receiver_id": "meowwwww.testnet", "amount": "1000000000000000000000000", "msg": "{\"match_id\": \"a team-b team-28/08/2024\", \"team\": \"Team1\"}"}' prepaid-gas '100.0 Tgas' attached-deposit '1 yoctoNEAR' sign-as pivortex.testnet network-config testnet sign-with-legacy-keychain send

get_match
near contract call-function as-read-only meowwwww.testnet get_match json-args '{"match_id": "a team-b team-28/08/2024"}' network-config testnet now

get_matches
near contract call-function as-read-only meowwwww.testnet get_matches json-args '{"from_index": null, "limit": null}' network-config testnet now

get_users_bets
near contract call-function as-read-only meowwwww.testnet get_users_bets json-args '{"bettor": "pivortex.testnet", "from_index": null, "limit": null}' network-config testnet now

get_bet 
near contract call-function as-read-only meowwwww.testnet get_bet json-args '{"match_id": "a team-b team-28/08/2024", "bet_id": "1"}' network-config testnet now




create another match
near contract call-function as-transaction meowwwww.testnet create_match json-args '{"game": "somegame", "team_1": "iamone", "team_2": "iamtwo", "in_odds_1": 1.4, "in_odds_2": 1.7, "date": "22/08/2024"}' prepaid-gas '100.0 Tgas' attached-deposit '0 NEAR' sign-as pivortex.testnet network-config testnet sign-with-legacy-keychain send
